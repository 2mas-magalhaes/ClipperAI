Agendamento Automático

Quando ativo, os vídeos são publicados como privados e agendados para saírem em intervalos regulares.

Como funciona:
  • O primeiro vídeo é agendado para daqui a 2 minutos
  • Os seguintes são espaçados pelo intervalo definido (exemplo: 2 em 2 horas)
  • Os vídeos ficam privados no YouTube até à hora agendada
  • Perfeito para distribuir conteúdo ao longo do dia automaticamente

Contas YouTube

As contas YouTube são geridas através dos canais. Adiciona um canal e associa o ficheiro de credenciais OAuth.

Como configurar:
  1. Vai a Google Cloud Console
  2. Cria um projeto e ativa a YouTube Data API v3
  3. Cria credenciais OAuth 2.0 (Desktop App)
  4. Faz download do client_secrets.json
  5. Adiciona o caminho do ficheiro ao criar o canal
